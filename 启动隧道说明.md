# 启动公网隧道指南

你的开发服务器已经在运行：
- 主应用: http://localhost:8080
- 设计师应用: http://localhost:8081
- 供应商应用: http://localhost:8082
- 用户应用: http://localhost:8084

## 方法1：使用 ngrok（推荐）

### 步骤1：下载 ngrok
访问：https://ngrok.com/download
下载 Windows 版本，解压到任意目录

### 步骤2：启动隧道
在 ngrok 所在目录打开命令行，运行：
```bash
ngrok http 8080
```

### 步骤3：获取公网地址
ngrok 会显示类似这样的地址：
```
Forwarding  https://xxxx-xx-xx-xx-xx.ngrok-free.app -> http://localhost:8080
```

复制 https 地址即可公网访问！

## 方法2：使用 Cloudflare Tunnel（免费，无需注册）

```bash
npx cloudflared tunnel --url http://localhost:8080
```

等待下载完成后会显示公网地址。

## 方法3：使用 VS Code 端口转发

1. 在 VS Code 中按 Ctrl+`打开终端
2. 点击"端口"标签
3. 点击"转发端口"
4. 输入 8080
5. 右键端口 -> "端口可见性" -> "公开"
6. 复制转发地址
